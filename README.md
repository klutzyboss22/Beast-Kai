# Beast Kai

Static Beast Kai website for GitHub Pages.

## Publish on GitHub Pages
1. Upload the `docs` folder and this `README.md` to your repository root.
2. In GitHub, open **Settings → Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select your main branch and `/docs` folder.
5. Save and wait for deployment.

The site is fully client-side and uses `.nojekyll` for GitHub Pages compatibility.
